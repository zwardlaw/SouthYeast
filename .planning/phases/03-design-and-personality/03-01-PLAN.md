    **MotionService.swift** -- Create in Services/:
    ```swift
    import CoreMotion
    import Observation

    @Observable
    @MainActor
    final class MotionService {
        var roll: Double = 0.0
        var pitch: Double = 0.0

        private let manager = CMMotionManager()

        func start() {
            guard manager.isDeviceMotionAvailable else { return }
            manager.deviceMotionUpdateInterval = 1.0 / 20.0  // 20Hz
            manager.startDeviceMotionUpdates(to: .main) { [weak self] data, error in
                guard let attitude = data?.attitude else { return }
                self?.roll = attitude.roll
                self?.pitch = attitude.pitch
            }
        }

        func stop() {
            manager.stopDeviceMotionUpdates()
        }
    }
    ```
    CRITICAL: Use `.main` OperationQueue, NOT a background queue. This keeps mutations on @MainActor safely (Swift 6 strict concurrency).

    **PizzaSliceNeedle.swift** -- Create in Views/. This file contains BOTH the Shape and the composed View:

    `PizzaSliceShape: Shape` -- draws a pizza slice sector pointing up (tip at center, crust arc at top):
    - `tipAngle = -90` (12 o'clock in SwiftUI coords where 0 = 3 o'clock)
    - `sliceHalfAngle = 18` (36-degree total arc width)
    - Path: move to center, line to left crust edge, addArc from left to right (clockwise: false), closeSubpath back to center
    - Use `addArc(center:radius:startAngle:endAngle:clockwise:)` with degrees converted to radians for the line endpoints

    `PizzaSliceNeedle: View` -- the composed needle view:
    - Takes `motionService: MotionService` and `isAligned: Bool` as parameters
    - Draws the PizzaSliceShape filled with a gradient or layered fills to suggest cheese/crust:
      - Base fill: Color.pizzaOrange (cheese)
      - Crust arc: overlay a thicker stroke at the outer edge in a darker warm brown or Color.pizzaRed
      - Optional: 3-4 small filled circles (Color.pizzaRed, varying sizes 4-8pt) positioned on the slice face as "pepperoni" for visual detail. Use `.overlay` with a ZStack of circles at fixed offset positions. Keep it simple -- this is illustration, not realism.
    - Apply two stacked `.rotation3DEffect()` calls for tilt parallax:
      - `.rotation3DEffect(.degrees(motionService.roll * 15), axis: (x: 0, y: 1, z: 0))` -- lean left/right
      - `.rotation3DEffect(.degrees(motionService.pitch * 15), axis: (x: -1, y: 0, z: 0))` -- lean forward/back
    - Alignment glow when `isAligned == true`:
      - `.shadow(color: Color.pizzaGold.opacity(isAligned ? 0.7 : 0), radius: 12)` -- warm glow
      - Animated pulse overlay: Circle stroke with `.pizzaGold`, `scaleEffect` and `opacity` animated with `.easeOut(duration: 0.8).repeatForever(autoreverses: false)` when aligned. Use `@State private var isPulsing` toggled to true via `.onChange(of: isAligned)` or `.onAppear` conditional.
      - Animate glow with `.animation(.easeInOut(duration: 0.3), value: isAligned)`
    - Size: `.frame(width: 180, height: 180)` as default, adjustable by caller
    - "Cooked vs cold" color temperature: shift the slice fill color between warm (Color.pizzaOrange when aligned) and desaturated/cooler (Color.pizzaOrange.opacity(0.5) blended with gray when misaligned). Use `foregroundStyle` or fill with a conditional color animated on `isAligned`. Keep it subtle -- this is aspirational.

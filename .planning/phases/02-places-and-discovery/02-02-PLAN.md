**CarouselView.swift — create new file with all carousel + card logic:**

**CarouselView (main struct):**
- `@Environment(AppState.self) private var appState`
- `@Environment(PlacesService.self) private var placesService`
- `@Environment(NetworkMonitor.self) private var networkMonitor`
- `@State private var scrollID: UUID?`
- `@State private var expandedID: UUID?`
- `@State private var isUserScrolling = true` — guards against programmatic scroll onChange
- Body uses a `GeometryReader` to calculate `cardWidth = proxy.size.width - 80` (40pt peek on each side)
- Conditional rendering based on state:
  - If `!networkMonitor.isConnected`: show offline error state (SF Symbol `wifi.slash`, "No internet connection", "Connect to find pizza nearby")
  - Else if `placesService.error == .noResults`: show empty state (SF Symbol `takeoutbag.and.cup.and.straw`, "No pizza nearby", "Try moving to a different area")
  - Else if `placesService.error != nil` (searchFailed): show error state (SF Symbol `exclamationmark.triangle`, "Something went wrong", "Pull down to try again")
  - Else if `placesService.isLoading && placesService.places.isEmpty`: show loading skeleton (2-3 placeholder rounded rects with `.redacted(reason: .placeholder)`)
  - Else: show the ScrollView carousel
- ScrollView `.horizontal`, `showsIndicators: false`:
  - `LazyHStack(spacing: 12)` containing `ForEach(placesService.places)` rendering `CardView` for each place
  - Each CardView gets `isExpanded: expandedID == place.id`, `onTap` closure toggling expandedID, `onLoadMore` closure (see below)
  - `.frame(width: cardWidth)` on each card, `.id(place.id)` for scroll tracking
  - `.padding(.horizontal, (totalWidth - cardWidth) / 2)` on LazyHStack to center first/last cards
  - `.scrollTargetLayout()` on LazyHStack
- ScrollView modifiers:
  - `.scrollTargetBehavior(.viewAligned)` for snapping
  - `.scrollPosition(id: $scrollID)`
  - `.onChange(of: scrollID)`: guard `isUserScrolling`, find matching place by ID, set `appState.selectedPlace = place`, collapse any expanded card (`expandedID = nil`)
- Load-more detection: In the ForEach, if the current place is among the last 3 places in the array, call `onAppear` that triggers `Task { await placesService.loadMore(userLocation: ...) }`. Get userLocation from appState or pass through environment — the CarouselView can read it from `@Environment(LocationService.self)`.
- Frame: the entire GeometryReader gets `.frame(height: expandedID != nil ? 280 : 140)` with `.animation(.spring(response: 0.35, dampingFraction: 0.75), value: expandedID)` for smooth expand/collapse of the carousel strip height
- Background: `.background(.ultraThinMaterial, in: RoundedRectangle(cornerRadius: 20))` on the outer container for the translucent overlay effect

**CardView (private struct in same file):**
- Properties: `let place: Place`, `let isExpanded: Bool`, `let onTap: () -> Void`
- `@AppStorage("preferredMapsApp") private var preferredApp: String = "apple"`
- `@AppStorage("hasChosenMapsApp") private var hasChosenMapsApp: Bool = false`
- `@State private var showMapsChoice = false`
- Collapsed content (always visible):
  - `Text(place.name).font(.headline)`
  - `Text(place.distanceDisplayString).font(.subheadline).foregroundStyle(.secondary)` — shows "X slices away"
- Expanded content (conditional on `isExpanded`):
  - `Divider()`
  - Address: `Text(place.address).font(.caption)` if not empty
  - Phone: `Text(place.phoneNumber).font(.caption)` if not nil
  - Website: `Link` to `place.websiteURL` if not nil, with `.font(.caption)`
  - Directions button: `Button("Get Directions")` with orange tint
    - On tap: if `!hasChosenMapsApp`, set `showMapsChoice = true`. Else call `openDirections(to: place, preferredApp: preferredApp)`
- Wrap in `VStack(alignment: .leading, spacing: 8)` with `.padding()`, `.frame(maxWidth: .infinity, alignment: .leading)`, `.background(.regularMaterial, in: RoundedRectangle(cornerRadius: 16))`
- `.onTapGesture { withAnimation(.spring(response: 0.35, dampingFraction: 0.75)) { onTap() } }`
- `.confirmationDialog("Choose Maps App", isPresented: $showMapsChoice)`:
  - Button "Google Maps": set `preferredApp = "google"`, `hasChosenMapsApp = true`, call `openDirections`
  - Button "Apple Maps": set `preferredApp = "apple"`, `hasChosenMapsApp = true`, call `openDirections`
  - Cancel button

**openDirections function (private, file scope or extension):**

- Walking directions by default (CONTEXT.md decision)
